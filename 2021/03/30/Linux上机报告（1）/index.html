实训1 Linux文件目录命令使用

一．实训目的

● 掌握Linux各类命令的使用方法。（正文：仿宋小四，文字行间距固定22磅，图片为1倍行间距，下同）

● 熟悉Linux操作环境。

● 掌握常用文件目录类命令。

● 掌握常用系统信息类命令

● 了解其他常用命令。

二．项目背景

现在有一台已经安装好Linux操作系统的主机，并且已经配置好基本的TCP/IP参数，能够通过网络连接局域网中或远程的主机。-台Linux服务器，能够提供FTP、Telnet和SSH连接。

三．实训内容

1.文件与目录管理

（1）查看根目录下有哪些内容?

cd /;ls

（2）进入/tmp目录，以自己的学号建一个目录，并进入该目录

cd /tmp;mkdir 201942308xxx;cd 201942308923

（3）显示目前所在的目录

pwd

（4）在当前目录下，建立权限为741的目录test1

mkdir -m 741 test1

（5）在目录tesl下建立目录test2/test3/test4

mkdir -p test2/test3/test4

（6）进入test2,删除目录test3/test4.

rmdir test3/test4

（9）复制目录/ete/下的内容到/tmp下。

cp -r /etc/ /tmp

(12)强制删除该文件。

rm -f aaa

(14)将/testl目录及其下面包含的所有文件删除。

rm -r /tmp/201942308xxx/test1

2.文件内容查阅、权限与文件查找

(4)使用more命令查看文件/var/log/ dmesg。

more /var/log/ dmesg

(5)使用less命令前后翻看文件/var/log/dmesg中的内容。

less /var/log/dmesg

(6)使用head命令查看文件/var/log/dmesg前20行。

head -n 20 /var/log/dmesg

(7)使用tail命令查看文件/var/log/dmesg后5行。

tail -n 5 /var/log/dmesg

(9)进人/tmp目录，将/root/. bashrc 复制成bashrc,复制完全的属性，检查其日期。

cp -a /root/.bashrc bashrc

3.文件查找

(1)查看文件/root/. bashrc的类型。

file /root/.bashrc

(2)查询ifconfig和cd命令的完整文件名。

有些时候最小化安装CentOS等Linux发行版的时候会默认不安装ifconfig等命令

which ifconfig;which ip

（3）只找出跟passwd有关的“说明文件”文件名。

whereis -m passwd

（4）找出系统中所有与passwd相关的文件名。

如果linux 最小化安装 会提示 -bash: locate: command not found 

locate passwd

（5）将过去3天前的24小时内有改动(mtime)的文件列出。

find / -mtime 3

（6）使用find查找当前目录下比/etc/hosts新的文件，并将查询的结果存储到hosts.new文件

find /home/用户名 -newer /etc/hosts >hosts.new

（8）查找系统中所有属于root的文件及不属于任何人的文件。

find / -user root; find / -nouser

（9）使用find查找/下面所有名称为包含man的文件。

find / -name "man*"

（10）找出系统中大于1 M的文件。

find / -size +1000k

4.设置文件权限

（3）对文件file1设置权限，使其他用户可以对此文件进行写操作。

chmod o+w file1

（4）查看设置结果。

ls -l

（5）取消同组用户对此文件的读取权限。查看设置结果。

chmod g= file1

(6)用数字形式为文件file1 设置权限，所有者可读、可写、可执行;其他用户和所属组用户只有读和执行的权限。设置完成后查看设置结果。

chmod 755 file1

（7）用数字形式更改文件file1设置权限，所有者只能读取此文件;其他用户和所属组用户没有权限。设置

完成后查看设置结果。

chmod 400 file1

(8）为其他用户添加写权限。查看设置结果。

chmod o+w file1

(9)回到上层目录，查看test的权限。

cd..;ll

(10）为其他用户添加对此目录的写权限。

chmod o+w test

5.改变所有者

(2)把目录test 及其下所有文件的所有者改为bin,所属组改为daemon。查看设置结果。

chown -R bin:daemon test

6.综合使用

（10）复制文件 profile到一个新文件profile. bak,作为备份。

cp profile profile.bak

(13)用tar命令把目录test打包。

tar cvf test.tar /home/violet

(14) 用gzip命令把打好的包进行压缩。

gzip test.tar

(15)把文件test. tar. gz改名为backup. tar. gz。

mv test.tar.gz backup.tar.gz

(16)把文件backup. tar.gz 移动到test目录下。

mv backup. tar.gz  test

(18)查找root用户自己主目录下的所有名为newfile的文件。

find /root -name newfile
