18.编写一个脚本，脚本名称命名为myshell。可判断数字的大小关系，文件是否存在、文件的类型以及文件权限，判断字符串的关系比较，同时将结果输出到屏幕上，具体要求如下：

[myt_4D@localhost ~]$ touch myshell.sh
[myt_4D@localhost ~]$ vim myshell.sh

（1） 依次输入1、2、3，分别进行数字比较、文件判断和字符串比较；
（2） 输入1后，要求输入两个数字，判断其大小关系；并输出其关系情况；

（3） 输入2后，要求输入一个文件名，判断其类型、文件权限，并输出其判断的信息；

（4） 输入3后，要求输入两个字符串，判断其关系，并输出其关系情况；
（5） 编写完脚本后，执行脚本验证其功能。

#!/bin/bash
echo enter you choice:
read  var

case $var in
        1)echo "please input two numbers:  "
        read a
        read b
        if [ $a -eq $b ]; then
                echo "$a = $b"
        elif [ $a -gt $b ]; then
                echo "$a > $b"
        else
                echo "$a < $b"
        fi;;
        2)echo -e "Please input a filename,I'll check the filename's type and its permissions.\n\n"
        read -p "Please input a fliename:" filename

        test -z $filename && echo "You must input a filename!Rember next time you mustn't input an empty file! " && exit 0
        test ! -e $filename && echo "The filename DO NOT exist!" && exit 0

        test -f $filename && filetype="regulare file"
        test -d $filename && filetype="directory"
        test -r $filename && perm="readable"
        test -w $filename && perm="$perm writeable"
        test -x $filename && perm="$perm executable"
        echo "The filename:$filename is a $filetype."
        echo "And the permissions is $perm.";;
        3)echo "please input two chars:"
        read STR
        read SUBSTR

        RESULT=$(echo "$STR" | grep $SUBSTR)

        if [ -n "$RESULT" ]; then
                echo -e "$SUBSTR in $STR.\n"
        else
                echo -e "$SUBSTR not in $STR.\n"
        fi;;
        *) echo wrong;;
esac
